# Bienvenue sur Steeplix ! 

## installation

git clone https://github.com/Mikael-W/nuxt3-Movie-App.git

Accéeder au dossier via un terminal et entrer la commande : npm install 


## Lancer le projet

entrer la commande: npm run start
